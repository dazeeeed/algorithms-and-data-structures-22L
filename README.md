# Algorithms and Data Structures - class 22L

## Lab 01 
Conversion of arabic alphabet to morse code.

Example: 
```bash
$ cat filename.txt
Ala ma kota
a kot ma Ale
a12b 3 c
```
Result:
```bash
$ python morse.py filename.txt
.- .-.. .- / -- .- / -.- --- - .-
.- / -.- --- - / -- .- / .- .-.. .
.- -... / -.-.
```
